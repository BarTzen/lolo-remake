# Lolo Remake

Recriação do jogo Adventures of Lolo (NES, 1989) desenvolvida como projeto acadêmico.

## Sobre o projeto

O jogo original é um puzzle de ação onde o jogador move o personagem Lolo por fases cheias de obstáculos e inimigos, coletando corações para abrir a saída. Este projeto recria a base do jogo usando Godot 4.

## Tecnologias

- [Godot 4](https://godotengine.org/) — engine
- GDScript — linguagem de script
- Sprites originais do NES (16×16 px)

## Estrutura do projeto

```
assets/       sprites e fontes
resources/    tilemaps e sprite frames gerados
scenes/       cenas do jogo (level_1, lolo, lala)
scripts/      scripts de movimento e utilitários
```

## Como rodar

1. Instale o Godot 4
2. Abra o projeto pelo `project.godot`
3. Pressione F5 para rodar
